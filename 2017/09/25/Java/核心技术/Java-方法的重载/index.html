Java-方法的重载

发表于 2017-09-25 | 分类于 Java

什么是方法的重载？

方法的重载就是在同一个类中允许同时存在一个以上的同名方法，只要这些方法的参数个数或类型不同即可。
方法的重载起源于构造方法。

构成方法重载的条件：

  1. 参数类型不同，构成重载。
  2. 参数个数不同，构成重载。
  3. 参数顺序不同，构成重载。

为什么要在项目中使用方法的重载？

方法的重载使得方法以统一的名称被管理，使程序代码有条理。
